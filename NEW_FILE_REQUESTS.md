# 새 파일 요청 (New File Requests)

모든 새 파일 요청은 이 문서에 제출해야 합니다.

## 제출 규칙

새 파일을 요청할 때는 **중복 기능 검색 결과**를 함께 기재합니다.

- **검색한 모든 위치**: 코드베이스에서 검색한 경로/키워드/파일 목록
- **검색 이유**: 해당 위치를 검색한 이유(기능 중복 가능성)
- **검색 결과 요약**: 유사·중복 기능 발견 여부 및 해당 위치
- **요청 내용**: 새로 만들 파일의 목적, 경로, 역할

## 제출 템플릿

```markdown
### [날짜] 새 파일 요청: [파일명 또는 기능명]

**요청 내용**
- (새 파일의 목적, 예상 경로, 역할)

**중복 검색**
| 검색 위치/키워드 | 검색 이유 | 결과 요약 |
|------------------|-----------|-----------|
| (예: lib/providers/) | (예: 동일 역할 프로바이더 존재 여부) | (발견/미발견, 해당 파일) |

**결론**
- (새 파일 생성 필요 여부 및 이유)
```

---

## 요청 이력

*(새 파일 요청이 들어오면 위 템플릿으로 이 섹션에 항목을 추가합니다)*
